Step 3: Creating Charts with D3.js

In this step, we'll create a simple bar chart using D3.js to visualize the average salary of different job roles in the job market. We'll assume that you have already fetched the relevant salary data from the API and stored it in a variable.

Charts with D3.js source link

Creating a Bar Chart

Below is the code snippet to create a simple bar chart using D3.js. This chart will display the average salary of different job roles in the job market. We'll use an SVG element to draw the chart and D3.js functions to bind the data to the DOM elements and set the properties for the chart.

Click on the titles to expand and see the code blocks.

																																				
																																					
        <div id="bar-chart"></div>

        // JavaScript code to create the bar chart using D3.js
        const salaryData = [
          { role: "Software Engineer", averageSalary: 80000 },
          { role: "Data Analyst", averageSalary: 70000 },
          { role: "Product Manager", averageSalary: 90000 },
          // Add more data points as needed
        ];

        const svgWidth = 600;
        const svgHeight = 400;
        const barPadding = 5;

        const svg = d3.select("#bar-chart")
          .append("svg")
          .attr("width", svgWidth)
          .attr("height", svgHeight);

        const barWidth = svgWidth / salaryData.length;

        const barChart = svg.selectAll("rect")
          .data(salaryData)
          .enter()
          .append("rect")
          .attr("y", (d) => svgHeight - d.averageSalary / 1000)
          .attr("height", (d) => d.averageSalary / 1000)
          .attr("width", barWidth - barPadding)
          .attr("transform", (d, i) => {
            const translate = [barWidth * i, 0];
            return "translate(" + translate + ")";
          })
          .style("fill", "steelblue");

The above code will create a bar chart displaying the average salaries of different job roles. The height of each bar represents the average salary, and the width of each bar is determined based on the number of data points.

After creating the bar chart, you can further customize it by adding axes, labels, and tooltips to enhance the user experience. With the bar chart created, let's move on to visualizing data on maps using D3.js.
