Unveiling Trends and Financial-Insights

Visualizes Various Investment Details

Each data point in the scatter plot represents a specific investment detail, and the chart displays these points on a coordinate grid.The scatter plot is used to compare different investment details based on their corresponding values. The x-axis represents the types of investment details (e.g., APY, CG Tax, Compound Interest, etc.), and the y-axis represents the values of those details. Each data point (circle) is positioned on the chart based on its type along the x-axis and its value along the y-axis.

Scatter Plot Chart

This is a Scatter Plot chart created using d3.js. It displays some sample data with visual representation of different investment parameters, allowing users to quickly understand and compare the values of each parameter. By hovering over the data points, users can see tooltips that provide specific information about each investment detail, such as its type and value.

Visualizes the Interest Rates of Different Countries obtained From an API

Each bar in the chart represents a country, and the height of the bar corresponds to the interest rate of that country. The purpose of the chart is to compare interest rates across different countries and identify the highest and lowest interest rates. The chart also allows for interactive elements, such as clicking on a bar to toggle its visibility, and hovering over a bar to see a tooltip with additional details. The fetched data is then transformed into a suitable format for visualization. The code extracts relevant information such as the country, rank, interest rate, and last updated date.

Bar Chart

This is a bar chart created using D3.js. It displays some sample data of countries in a ranked order based on their interest rates. Enabling users to perform the following actions, Hovering over a bar displays a tooltip with additional details about the country, rank, interest rate, and last updated date. Clicking on a bar toggles its visibility. When clicked, a bar becomes red and disappears from the chart.
