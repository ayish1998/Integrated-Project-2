Data Visualization: Practical How-Tos for Presentation and Analysis

Overview

Introduction

Welcome to this comprehensive tutorial on harnessing the power of live APIs to effortlessly fetch, display, and visualize weather data! Over the course of this step-by-step guide, we will delve into the world of web development using Vanilla JavaScript, where we'll master the art of data retrieval from finance APIs. We'll then seamlessly process and manipulate this data using cutting-edge techniques.

We'll delve into mesmerizing data visualization with D3.js, crafting appealing bar charts to showcase company details. Color-coded bars with interactive tooltips will enrich the experience. Next, we'll create captivating bubble charts, displaying financial data like market capitalization and P/E ratios. Interactive bubbles reveal insights when hovered. Our tutorial delves deep into crafting functional data tables. Learn to structure, populate, and add sorting for easy data navigation.

File Structure

The financial data landing page harnesses the potential of the financial.js file as its cornerstone. This pivotal file serves as the entry point, seamlessly linking with the main financial.html page to orchestrate the immersive financial data experience. Through strategic utilization of HTML5, CSS3, and Bootstrap 5.0, this page sets the stage for a dynamic journey into the world of finance.

Upon arrival, users are greeted by six informative cards, each acting as a gateway to dedicated pages abundant with relevant financial information and associated visualizations. The financial.js file's orchestration facilitates this seamless navigation, ensuring that users effortlessly access their desired insights. For captivating visualizations, other vital JavaScript files reside in the broader JS folder.

Step 2: Filtering The Data

After fetching the data, we apply a filtering process to narrow down the stocks based on specific criteria. These criteria help us identify stocks with promising potential for growth. The filterStocks function calculates the earnings growth rate, PE ratio, and PEG ratio for each stock and then filters out those stocks that meet certain conditions. This step ensures that we're focusing on stocks that align with our investment strategy. Here's the code to filter the stocks:

Click on the titles to expand and see the code blocks.

                        
                          function filterStocks(data) {
                            const filteredStocks = data.quotes.filter(stock => {
                              const earningsGrowthRate = (stock.epsTrailingTwelveMonths / stock.epsForward - 1) * 100;
                              const peRatio = stock.trailingPE;
                              const pegRatio = peRatio / earningsGrowthRate;
                              return earningsGrowthRate > 25 && peRatio < 20 && pegRatio < 1.5;
                            });
                            return filteredStocks;
                          }

Step 5: Adding Bars, Bubbles with Tooltips

Continuing from the previous step, we'll create a bar chart using D3.js to visualize the earnings growth rate of each stock. We'll set up scales for the axes, append SVG elements, and handle mouse events for interactivity.The x and y scales are defined to map the stock symbols and earnings growth rates to the chart's coordinates. These scales ensure that the data is appropriately positioned and scaled within the chart area.

Click on the titles to expand and see the code blocks.

                      

                        // Add bars to the bar chart
                        svg.selectAll(".bar")
                          .data(filteredData)
                          .enter()
                          .append("rect")
                          .attr("class", "bar")
                          .attr("x", stock => x(stock.symbol))
                          .attr("y", stock => y((stock.epsTrailingTwelveMonths / stock.epsForward - 1) * 100))
                          .attr("width", x.bandwidth())
                          .attr("height", stock => height - y((stock.epsTrailingTwelveMonths / stock.epsForward - 1) * 100))
                          .attr("fill", "steelblue");                
                          .on("mouseover", (event, stock) => handleMouseOver(event, stock))
                          .on("mouseout", handleMouseOut);

                          // Mouseover event handler for the bars
                          function handleMouseOver(event, stock) {
                           tooltip.transition()
                           .duration(200)
                           .style("opacity", 0.9);

                          const earningsGrowthRate = (stock.epsTrailingTwelveMonths / stock.epsForward - 1) * 100;
                          const peRatio = stock.trailingPE;
                          const pegRatio = peRatio / earningsGrowthRate;
                        
                          tooltip.html(
                            `${stock.symbol}

                              Earnings Growth Rate: ${earningsGrowthRate.toFixed(2)}%

                              PE Ratio: ${peRatio.toFixed(2)}

                              PEG Ratio: ${pegRatio.toFixed(2)}`
                          )
                            .style("left", event.pageX + "px")
                            .style("top", (event.pageY - 28) + "px");
                        }
                        
                        // Mouseout event handler for the bars
                        function handleMouseOut() {
                          tooltip.transition()
                            .duration(500)
                            .style("opacity", 0);
                        }

This code defines two functions: 'handleMouseOver' and 'handleMouseOut'. The 'handleMouseOver' function displays a custom tooltip with information about the hovered bubble's stock. The 'handleMouseOut' function hides the tooltip when the mouse leaves the bubble.
